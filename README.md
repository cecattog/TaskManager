# TaskManager


A simple iOS task management app built with SwiftUI and Firebase.

## Features
- User authentication (sign up / log in)
- Create, complete, and delete tasks
- Real-time sync with Firestore
- Loading indicators for network operations

## Technologies
- SwiftUI
- Firebase Auth
- Firestore

## Requirements
- iOS 17.0+
- Xcode 15.0+
- Firebase account (free tier)

## Setup
1. Clone the repository
2. Add your own `GoogleService-Info.plist`
3. Enable Email/Password Auth and Firestore in Firebase Console
